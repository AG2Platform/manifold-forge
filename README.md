# Apple-Inspired Dashboard

A general-purpose, ambient dashboard inspired by Apple's design language. Built with React, Vite, TypeScript, Tailwind CSS, Framer Motion, and Recharts.

## Features
- Glassmorphic cards with backdrop blur
- Animated counters with spring physics
- Sample line chart (activity/usage over time)
- Notifications panel with icons
- Quick actions area with circular icon buttons
- Responsive layout for desktop and mobile
- San Franciscoâ€“style system font
- Subtle gradients and muted accent glows
- Smooth, elegant transitions and micro-interactions
- Professional, minimalist color palette (light mode, soft gray backgrounds)

## Tech Stack
- React 18 + TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React (icons)
- Recharts (charts)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
